# JARVIS AI Assistant

A personal AI assistant inspired by JARVIS from Iron Man.

## Current Features

- Text-to-speech responses
- Command routing system
- Open Visual Studio Code
- Open GitHub
- Report the current time
- Graceful shutdown

## Planned Features

- Voice recognition
- Wake word detection
- AI integration
- Memory system
- Flask dashboard
- Local automation
- AWS deployment
- Local LLM support

## Technologies

- Python
- Git / GitHub
- pyttsx3
- VS Code
